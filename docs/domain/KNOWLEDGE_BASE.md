# KNOWLEDGE_BASE

Base de conocimiento consolidada a partir de los archivos del paquete `CONOCIMIENTOS.zip`.

## Alcance

- Esta versión transforma los archivos fuente en una referencia única en Markdown.
- Se privilegia la **clasificación temática**, el **resumen operativo** y un **extracto verificable** por archivo.
- Cuando un archivo es duplicado, imagen o formato con extracción limitada, se deja señalado explícitamente.

## Criterio rector de uso

El archivo `DIRECTRIZ` establece un marco de análisis ejecutivo y preventivo para casos en México. En cualquier uso posterior de esta base conviene mantener estas etiquetas:

- `[HECHO CONFIRMADO]`
- `[INFORMACIÓN FALTANTE]`
- `[SUPUESTO]`
- `[RIESGO]`
- `[RECOMENDACIÓN]`

Además, la directriz sugiere resolver asuntos en 5 fases: diagnóstico integral, revisión de cumplimiento/materialidad, matriz de riesgos, escenarios de resolución y checklist de ejecución.

## Mapa temático

- **Seguridad social, nómina y cumplimiento laboral:** IMSS, SBC, INFONAVIT, SIROC, REPSE, RCV/AFORE, contratos colectivos y ejercicios prácticos.
- **Contabilidad, estados financieros y fiscal:** contabilidad electrónica, NIF, costos, activo fijo, reexpresión, estados comparativos y cierre fiscal.
- **Presupuestos, indicadores y administración:** presupuestos, indicadores financieros, Balanced Scorecard y proceso administrativo.
- **Material visual:** imágenes de apoyo no procesadas con OCR en esta versión.
